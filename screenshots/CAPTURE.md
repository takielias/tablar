# Capturing Tablar screenshots

These screenshots illustrate Tablar on a fresh Laravel 13 install with Vite 8.
They are regenerated by the Playwright capture script in Phase 8 of the
revamp (E2E DDEV demo at `/home/taki/laravel/revamp-demo`).

## Required artefacts

| Filename                  | View                                 |
|---------------------------|--------------------------------------|
| `welcome-light.png`       | `/` welcome page, light theme        |
| `welcome-dark.png`        | `/` welcome page, dark theme         |
| `login.png`               | `/login` form                        |
| `dashboard.png`           | `/home` empty-state dashboard        |
| `crud-list.png`           | sample CRUD index after `make:crud`  |

Capture at 1440×900, light + dark variants where indicated. Save as
`.png` with a max width of 1600px after retina-scaling.

## Why local files

Earlier README versions referenced external URLs (banners.beyondco.de,
github.com user-attachment URLs). External hosts decay; local artefacts
in `screenshots/` are version-controlled alongside the package and stay
consistent with the README on every release.
